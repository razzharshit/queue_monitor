# `@queue-monitor/shared`

Public TypeScript event contracts and ingestion validation used by Queue Monitor SDKs and services. It supports UUID trace IDs and non-zero 32-character W3C/OpenTelemetry trace IDs.

```ts
import { validateEvent, type IngestEvent } from "@queue-monitor/shared";
```

This package follows Semantic Versioning and ships ESM, CommonJS, declarations, and source maps under the MIT license.
